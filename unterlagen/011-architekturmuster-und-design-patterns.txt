Grundidee der folgenden Architekturmuster ist folgende:
Man möchte den Code bzw. die Logik für die Präsentation (
Darstellung der Fenster, Reaktion auf Eingabe durch Benutzer, Fenstersteuerung etc.)
von Code der Geschäftslogik (z.B. Lohnabrechnung, Steuerklassen, Sozialabgaben, )

Model-View-Controller
Model-View-Presenter
Model-ViewModel-View (MVVM)

- Abstract Factory

- State
